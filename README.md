# StoreStock

StoreStock is a database group project designed to manage product inventory,
stock levels, suppliers, and sales records.

## Project Scope
- Inventory management
- Stock tracking
- Supplier relationships
- Sales data handling

## Technologies Used
- SQL (MySQL / PostgreSQL / Oracle — specify yours)
- ER Diagram
- Relational Database Design

## Files Structure
- `/database` – SQL schema and sample data
- `/diagrams` – ER diagrams and design visuals
- `/report` – Project documentation
